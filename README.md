## Modern Full-Stack Developer Portfolio 

A sleek, dark-themed responsive portfolio website designed for Full-Stack Engineers. This project features a modern "Syne" and "Space Mono" typography pairing, a custom pixel-art aesthetic, and a fully responsive grid layout.

## The Stack used

This is a lightweight, high-performance frontend implementation using:

* **HTML5** – Semantic structure for SEO and accessibility.
* **CSS3** – Custom properties (variables), Flexbox, and CSS Grid for a modern layout.
* **Google Fonts** – Integrated 'Syne' for bold headings and 'Space Mono' for a technical feel.
* **SVG Art** – Custom pixel-art illustrations rendered directly in code.

## Key Features

* **Responsive Design:** Fully optimized for Desktop, Tablet, and Mobile devices.
* **Sticky Navigation:** Glassmorphism effect with `backdrop-filter` for a premium feel.
* **Dynamic Sections:** * **Hero:** Impactful messaging with a pulsing "availability" badge.
    * **About:** Clean layout featuring custom SVG pixel illustrations.
    * **Skills Grid:** Categorized technical stack tags.
    * **Experience Timeline:** Vertical roadmap of professional history.
    * **Project Showcase:** Hover-effect cards with direct links to GitHub/Demos.
* **Contact System:** Integrated UI for inquiries and social connectivity.
